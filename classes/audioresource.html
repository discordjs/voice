read

  • read(): null | Buffer
    • Defined in src/audio/AudioResource.ts:109

    Attempts to read an Opus packet from the audio resource. If a packet is available, the playbackDuration is incremented.

    internal
    remarks

    It is advisable to check that the playStream is readable before calling this method. While no runtime errors will be thrown, you should check that the resource is still available before attempting to read from it.
